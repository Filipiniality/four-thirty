class Entry {
 public:
  bool reference = false; // true whenever this block as accessed. false by 2nd chance alg.
  bool dirty = false; // true whenever this block is written. false upon a writeback.
  int pNumber = -1; // contains the disk block number. -1 if this entry has no block.
};

Block-read algo:
- Sequentially search all cache slots
- If the corresponding entry has been found in the cache, read the contents from the cache (cache hit)
- If not found (not cached), find a free block entry in the cache and load the data from the disk to this cache block
- Do 2nd chance algo
- If this victim is dirty, you must first write back its contents to the disk first, and thereafter load the new page from the disk into this cache block.
- Do not forget to set the reference bit.

Block-write algo:
- Sequentially search all cache slots
- If the corresponding entry has been found in the cache, overwrite new data to this cache block. (is now dirty)
- Otherwise, find an empty block entry in the cache and write the data to this cache block. (is now dirty)
- 2nd chance algo
- If this victim is dirty, you must first write back its contents to the disk and thereafter write new data into this cache block. Do not forget to set the reference bit.

// read a requested page from disk
lines[i] = (char*)mmap( lines[i], page_len, PROT_READ | PROT_WRITE, MAP_SHARED,
